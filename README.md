# RateMyBite

RateMyBite is a full-stack web application that allows users to discover restaurants through an interactive map and rate their dining experiences. Instead of scrolling through long lists, users can visually explore nearby food spots, read reviews, and share their own. The goal is to make food discovery more intuitive, engaging, and community-driven.

## Features
- Interactive map displaying nearby restaurants
- User reviews, star ratings, and food photos
- Search and filtering by distance, rating, and cuisine
- Restaurant detail pages with menus, hours, and contact info
- User accounts, favorites, and review history
- Admin dashboard for managing restaurant listings
